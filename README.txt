--------------------------Storygen--------------------------
1. Running the code

To run this program you will need to run the storyGenerator class's main method. This will open a GUI where you can press the generate button to generate a story. 

You can simply open the project in a IDE such as intelliJ and press the run button in the storyGenerator class to run the main method.

The class is located in StoryGen/src/storyGenerator.java.

------------------------------------------------------------

2. Saving location

When you save a story it is saved as "storyOutput.txt" in the main StoryGen folder.
